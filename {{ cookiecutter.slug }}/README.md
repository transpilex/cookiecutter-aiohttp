## Quick Start

**Install uv**
```bash
pip install uv
```

**Sync environment (install deps into .venv)**
```bash
uv sync
```

**Run FastAPI (development)**
```bash
uv run python run.py
```

**Assets Compilation**

In separate terminal run following commands:
```bash
npm i
```

```bash
npm run dev
```

**Open browser**

- App: http://127.0.0.1:8080
